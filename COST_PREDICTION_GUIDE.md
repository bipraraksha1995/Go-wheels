## Key Features

### 1. Optional Input
- Sellers are NOT required to enter manual values
- System works perfectly with or without manual input

### 2. Visual Indicators
- **âœï¸ Seller Provided** - Manual values entered by seller
- **ğŸ¤– Auto-Calculated** - System calculated based on formula

### 3. Transparency
- Buyers can see the source of cost information
- Builds trust between buyers and sellers

### 4. Flexibility
- Sellers with accurate cost data can provide it
- Sellers without data can rely on auto-calculation

### 5. Smart Fallback
- If any manual field is empty, auto-calculation is used
- No partial manual data - it's all or nothing
